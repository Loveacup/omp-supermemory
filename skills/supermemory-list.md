# Supermemory List

List recent memories from your Supermemory knowledge base.

## When to Use

- User asks "what do you remember about me?"
- User says "show what's saved" or "list memories"

## How to List

Run:
```
node ~/.omp/supermemory/search-memory.js "" --scope user
```

Use `--scope project` for project-specific memories.
